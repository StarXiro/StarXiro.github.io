# 标题

# 方法一：

标题按照 # 的个数进行排列，最多支持六级。

示例：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
# 方法二：

在标题文本下方的行输入任意数量的 = （一级标题）或 - （二级标题）字符。

示例：

一级标题
=

二级标题
-----

# 换行

# 方法一：

使用 双回车 换行。

示例：

这是一句话。

这是另一句话。
# 方法二：

使用 双空格+回车 换行。

示例：

这是一句话。  
上面一段话的结尾有两个空格。
# 方法三：

使用 html 语言换行标签 <br> :

示例：

原神，<br>启动！

# 字体相关

# - 加粗

使用 ** 包裹文字。

# - 斜体

使用 * 包裹文字。

# - 斜体加粗

使用 *** 包裹文字。

# - 删除线

使用 ~~ 包裹文字。

示例：

**这是加粗的文字**
*这是斜体的文字*
***这是斜体加粗的文字***
~~这是加删除线的文字~~

# 分割线

使用大于等于三个的 -、*或者_ 进行分割。

示例：

---
____
*****

# 缩进字符

&nbsp; 缩进 1/2 字符（1/4 中文）

&ensp; 缩进 1 字符（1/2 中文）

&emsp; 缩进 2 字符（1 中文）

# 列表

# 无序列表

使用 -、+或*加空格 插入。

示例：

- 内容1
+ 内容2
* 内容3
# 有序列表

使用 数字+. 插入。

示例：

1.内容1
2.内容2
3.内容3
